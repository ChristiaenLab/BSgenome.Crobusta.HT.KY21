# BSgenome.Crobusta.HT.KY21
A BSgenome R package for the Hoya T-line KY2021 Ciona robusta genome. This script downloads the genome and installs it as a BSgenome package, which can be used for motif matching in R. Installing (but not running) the package requires the `optparse` package.
# Usage
```bash
cd BSgenome.Crobusta.HT.KY21
make
```
